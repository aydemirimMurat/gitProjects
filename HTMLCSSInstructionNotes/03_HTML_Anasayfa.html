id Attribute:

  • HTML element leri için unique id oluşturmaya yarar.
  • id attribute un değeri, o html dokümani içerisinde unique olmali
  • CSS ve JavaScript, belirli bir elementle ilgili (style/select) yapmak için id leri kullanir.
  • CSS ile belirli bir elemente "#ElementId" ile ulaşilabilir.
  • JavaScript: JavaScript: belirli bir elemente getElemntById() ile ulaşir.
  • Id değerleri case sensitive dir.

Class Attribute:

Element ler için class (container) belirler.
CSS ve JavaScript ten, class Attribute u kullanarak belirli bir class name e sahip element lere ulaşmak ve işlem yapmak için kullanilir.

  • Bir element için birden fazla class belirlenebilir.
  • Class attribute u global attribute tur. Yani herhangi bir element veya tüm elementler ile kullanilabilir.
  • Farkli elementler ayni class attribute una sahip olabilir.
  • CSS ve JavaScript bu class attribute larini kullanarak belirli elementlere ulaşabilir.
  • CSS ten belirli bir class name e ".ClassName" ile ulaşilabilir.
  • JavaScript, belirli class name e sahip elementlere getElementByClassName() ile ulaşabilir.
  • class değerleri case sensitive dir.
